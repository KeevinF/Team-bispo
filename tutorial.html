📸 Como Gerenciar as Fotos

Guia rápido e fácil para adicionar transformações ao seu site

1

Acesse a página de Admin

Abra seu navegador e vá para:

http://localhost:5500/admin.html

*(Substitua o endereço pelo do seu site se for diferente)*

2

Selecione uma foto

Clique no botão "📁 Selecione uma foto para adicionar"

Uma janela vai abrir no seu computador. Escolha uma foto que quer adicionar.

Formatos aceitos: JPG, PNG, GIF

3

Confirme o upload

Após escolher a foto, você verá um preview dela na página.

Clique no botão "Adicionar Foto" para confirmar.

Uma mensagem ✓ Foto adicionada com sucesso! vai aparecer.

4

Veja no site

Volte ao seu site principal e recarregue a página (tecla F5).

A foto já deve estar lá! Use as setas ◀ ▶ para navegar pelas fotos.

5

Repetir para mais fotos

Volte ao admin e repita os passos 2-3 para adicionar quantas fotos quiser.

Pode adicionar 5, 10, 50 fotos! O site vai mostrar uma por vez.

🗑️ Como Remover uma Foto

  1. Acesse a página de admin
  2. Procure a foto que quer remover na lista
  3. Clique no botão vermelho "Remover"
  4. Confirme a remoção
  5. Pronto! A foto sumiu do site

📋 Informações Importantes

✅ O que funciona

✓ JPG, PNG, GIF

✓ Até 5MB por foto

✓ Sem limite de quantidade

✓ Qualquer tamanho

❌ O que NÃO funciona

✗ Vídeos

✗ Arquivos de áudio

✗ Imagens muito pesadas (>5MB)

💡 Dicas de Foto

• Boa iluminação

• Foco na transformação

• Sem marca d'água

• Altura: qualquer uma

🔧 Resolução

Mínimo: 400px de largura

Máximo: sem limite

Recomendado: 600-800px

💡 DICA: Se a foto ficar de cabeça para baixo ou virada, edite a imagem no seu computador ANTES de adicionar. Rotacione a foto e salve, depois tente novamente.
⚠️ IMPORTANTE: As fotos ficam salvas no navegador do seu computador. Se você limpar o cache/histórico, as fotos podem ser deletadas. Faça backup das imagens!
