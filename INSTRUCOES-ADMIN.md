# 📸 Como Adicionar Fotos das Transformações

## ✅ Passo a Passo

### **1️⃣ ACESSAR A PÁGINA DE ADMIN**
   - Abra seu navegador
   - Digite na barra de endereços:
     ```
     http://localhost:5500/admin.html
     ```
     *(Substitua `localhost:5500` pelo endereço do seu site)*

### **2️⃣ VOCÊ VERÁ ESTA TELA**
   - Uma página com o título "📸 Gerenciar Transformações"
   - Um botão "📁 Selecione uma foto para adicionar"
   - Uma lista de fotos já adicionadas

### **3️⃣ ADICIONAR UMA FOTO**
   - Clique no botão **"📁 Selecione uma foto para adicionar"**
   - Uma janela vai abrir pedindo para escolher uma imagem do seu computador
   - Escolha uma foto (JPG, PNG, GIF)
   - Clique em **"Abrir"**

### **4️⃣ CONFIRMAR O UPLOAD**
   - Você vai ver a foto aparecer no preview
   - Clique no botão **"Adicionar Foto"**
   - Uma mensagem verde dirá: "✓ Foto adicionada com sucesso!"

### **5️⃣ PRONTO!**
   - A foto já aparece no seu site automaticamente
   - Você pode adicionar quantas fotos quiser
   - Use as setas ◀ ▶ no site para navegar pelas fotos

---

## 🗑️ Remover uma Foto

1. Acesse `/admin.html` novamente
2. Veja a lista de fotos adicionadas
3. Clique no botão vermelho **"Remover"** ao lado da foto
4. Confirme que deseja remover
5. Pronto! A foto some do site

---

## ❌ Resolver Problemas

### **As fotos não aparecem no site?**
- Confirme que você clicou em **"Adicionar Foto"** (não é automático)
- Verifique se a mensagem verde apareceu
- Recarregue a página do site (F5)

### **A foto está de cabeça para baixo?**
- Edite a foto no seu computador antes de adicionar
- Rotacione a imagem e salve
- Depois adicione a foto corrigida

### **Qual é o tamanho certo para a foto?**
- Largura: mínimo 400px
- Altura: qualquer uma (não precisa ser quadrada)
- Tamanho do arquivo: máximo 5MB

---

## 💡 Dicas

✓ Use fotos com boa iluminação  
✓ Foque no corpo (mostre a transformação)  
✓ Pode adicionar vídeos? Não, só imagens  
✓ As fotos ficam salvas no navegador (não é nuvem)
